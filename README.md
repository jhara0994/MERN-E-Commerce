## Description:
### This is a full stack appplication using MERN and Context API to produce a store you can shop through and dashboards for the buyer and the seller.
## Installation:
### To run this application locally, clone the repository from Github. Once the code files are available, run `npm i` in your terminal to install all dependencies. 
## Usage:
### This application is intended to be used as an e-commerce store that can have multiple sellers and buyers. It is built as an art store, but can be modified for your individual store needs.
